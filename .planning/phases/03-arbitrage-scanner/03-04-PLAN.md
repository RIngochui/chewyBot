  <behavior>
    - Group normalized records by (event_id, market_type) — only compare same event + market
    - For each group, find best decimal_odds per selection_name across all books
    - If sum(1/best_odds for each selection) < 1.0 → arb exists
    - arb_pct = (1.0 - sum_of_implied) * 100
    - Filter: only yield ArbSignal if arb_pct >= min_arb_pct
    - Stake calculation (for 2-outcome markets):
        total_implied = sum(1/odds for odds in best_odds_per_side)
        stake_a = bankroll * (1/odds_a) / total_implied  — rounded to 2 decimals
        stake_b = bankroll * (1/odds_b) / total_implied  — rounded to 2 decimals
        estimated_profit = round(bankroll * arb_pct / 100, 2)
    - Skips markets with only 1 selection (can't arb a single-outcome market)
    - market_key in ArbSignal: use the market_key from side A's NormalizedOdds record
    - For Lakers 2.20 (FanDuel) vs Warriors 2.25 (DraftKings): sum_implied = 1/2.20 + 1/2.25 = 0.4545 + 0.4444 = 0.899 → arb_pct = 10.1%
    - detect_arb([...lakers_warriors_records...], min_arb_pct=0.5, bankroll=100.0) returns list with 1 ArbSignal, arb_pct ≈ 10.1

    For detect_ev():
    - Group records by (event_id, market_type)
    - For each group, compute fair probabilities via no_vig_probability() across all offered decimal odds for that market
    - For each selection, compare each book's offered_decimal against fair_prob:
        ev_pct = ((offered_decimal * fair_prob) - 1) * 100
    - Yield EVSignal if ev_pct >= min_ev_pct
    - Uses first NormalizedOdds record's event_name/sport/market_type for the signal
  </behavior>

    detect_arb algorithm:
    ```python
    from itertools import groupby
    from collections import defaultdict

    # Group by (event_id, market_type)
    groups: dict[tuple, list[NormalizedOdds]] = defaultdict(list)
    for rec in normalized:
        groups[(rec.event_id, rec.market_type)].append(rec)

    for (event_id, market_type), records in groups.items():
        # Best odds per selection across all books
        best: dict[str, tuple[float, str, str]] = {}  # selection -> (odds, book, market_key)
        for rec in records:
            if rec.selection_name not in best or rec.decimal_odds > best[rec.selection_name][0]:
                best[rec.selection_name] = (rec.decimal_odds, rec.book_name, rec.market_key)

        if len(best) < 2:
            continue

        sum_implied = sum(1.0 / odds for odds, _, _ in best.values())
        if sum_implied >= 1.0:
            continue

        arb_pct = (1.0 - sum_implied) * 100
        if arb_pct < min_arb_pct:
            continue

        selections = list(best.items())
        sel_a, (odds_a, book_a, mkey) = selections[0]
        sel_b, (odds_b, book_b, _)    = selections[1]

        stake_a = round(bankroll * (1/odds_a) / sum_implied, 2)
        stake_b = round(bankroll * (1/odds_b) / sum_implied, 2)
        profit  = round(bankroll * arb_pct / 100, 2)

        signals.append(ArbSignal(
            market_key=mkey, event_name=records[0].event_name,
            sport=records[0].sport, market_type=market_type,
            arb_pct=round(arb_pct, 4), stake_side_a=stake_a, stake_side_b=stake_b,
            estimated_profit=profit, book_a=book_a, book_b=book_b,
            odds_a=odds_a, odds_b=odds_b, selection_a=sel_a, selection_b=sel_b,
        ))
    ```
