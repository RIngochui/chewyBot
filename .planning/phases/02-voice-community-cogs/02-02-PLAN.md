**Slash command /seek [seconds]** (MUS-10):
```python
@app_commands.command(name="seek", description="Seek to a position in the current track")
@app_commands.describe(seconds="Position in seconds to seek to")
async def seek(self, interaction: discord.Interaction, seconds: int) -> None:
```
Implementation:
1. Check vc is connected and `state['is_playing']`. If not, respond ephemeral "Nothing is playing."
2. Check `0 <= seconds <= song['duration']`. If out of range, respond ephemeral "Position out of range."
3. `vc.stop()` — this triggers the `after_play` callback which would normally advance the index.
   To prevent advancing, use a `state['seek_offset']` flag: set `state['seek_offset'] = seconds`
   before calling `vc.stop()`, then in `_play_next`, check for seek_offset and pass it to FFmpegPCMAudio:
   ```python
   offset = state.pop('seek_offset', None)
   if offset is not None:
       before_opts = f"-reconnect 1 -reconnect_streamed 1 -reconnect_delay_max 5 -ss {offset}"
   else:
       before_opts = "-reconnect 1 -reconnect_streamed 1 -reconnect_delay_max 5"
   source = discord.FFmpegPCMAudio(song['url'], before_options=before_opts, options='-vn')
   ```
   Also in `after_play` callback: if `state.get('seek_offset')` is not None (checked BEFORE pop),
   do NOT advance `current_index` — keep it pointing at the same song.
   After seek: reset `state['start_time']` to `datetime.datetime.utcnow() - datetime.timedelta(seconds=seconds)`
   so progress bar reflects the seek position.
4. Respond: embed f"Seeked to {_format_duration(seconds)}" (color=EMBED_COLOR).

**Note on seek approach:** After `vc.stop()`, the `after_play` runs in a thread via `asyncio.run_coroutine_threadsafe`. The seek_offset flag must be set BEFORE `vc.stop()`. In `after_play`, check the flag and skip index advancement if set. `_play_next` pops the flag and uses it in FFmpegPCMAudio before_options.


2. Separate currently-playing song (if any) from rest:
   ```python
   current_song = state['queue'][state['current_index']] if state['is_playing'] else None
   remaining = [s for i, s in enumerate(state['queue']) if i != state['current_index'] or not state['is_playing']]
   import random
   random.shuffle(remaining)
   if current_song:
   ```

```python
async def _log_embed(
    self,
    guild: discord.Guild,
    event_type: str,
    song: dict | None = None,
) -> None:
    channel = self.bot.get_channel(config.LOG_CHANNEL_ID)
    if channel is None:
        logger.warning("Log channel %d not found", config.LOG_CHANNEL_ID)
        return
    embed = discord.Embed(color=EMBED_COLOR)
    if event_type == "song_start" and song:
        embed.title = f"Now Playing: {song['title']}"
        embed.url = song.get('webpage_url', '')
        embed.set_thumbnail(url=song.get('thumbnail', ''))
        embed.add_field(name="Duration", value=_format_duration(song.get('duration', 0)))
        embed.add_field(name="Requested By", value=song.get('requester', 'Unknown'))
    elif event_type == "playlist_added" and song:
        embed.title = f"Playlist Added: {song.get('name', 'Unknown Playlist')}"
        embed.add_field(name="Tracks", value=str(song.get('count', 0)))
        if song.get('thumbnail'):
            embed.set_thumbnail(url=song['thumbnail'])
    elif event_type == "queue_end":
        embed.title = "Queue Finished"
        embed.description = "No more songs in queue."
    else:
        return
    await channel.send(embed=embed)  # type: ignore[union-attr]
```
